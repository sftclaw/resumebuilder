How It Works

1

Export from LinkedIn

Go to Settings → Data Privacy → Get a copy of your data. Select all data and download the ZIP file.

2

Drop & Parse

Drag the ZIP file onto our builder. We'll instantly parse your profile, experience, education, skills, and more.

3

Customize

Edit your info, pick a theme, toggle sections on/off. See changes live in the preview.

4

Download & Deploy

Download a single HTML file with everything baked in. Upload to GitHub Pages, Netlify, or host it wherever you want.
